雅安市纵横计算机网络有限公司简介
雅安市纵横计算机网络有限公司成立于1999年，注册资本金为人民币30万元整，是一家专注于信息技术产品与服务的专业化企业。作为一家具有深厚行业背景的IT服务商，公司致力于为政府、军队、企业及事业单位提供全方位的IT解决方案，涵盖以下领域：
• 计算机硬件销售
• 软件开发及设计
• 网络安全
• 系统集成
• 弱电工程
• 监控及报警系统
• 视频会议及投影产品
• 办公设备及外设
• 数码产品维修
• 打印机、复印机维护
• 其他网络产品维修服务
公司秉持“诚信为本”的理念，以“您的满意，是我们最大的荣誉”作为公司运营的核心信念。经过十余年的稳健发展，公司现有员工18人，其中包括：
销售团队：6人
物流管理：1人
财务部：1人
商务部门：2人
技术支持团队：8人
公司配备了完善的管理体系，包括标准化的公司管理制度、专业的售后服务体系以及规范的财务管理制度。同时，公司在雅安市区设有：
联想电脑专业销售店面
专业维修服务店面
大型库房设施
所有技术人员均已通过厂商专业培训，拥有一定的实践经验和技术能力，为客户提供专业、可靠的技术支持。
凭借优质的产品和用心的服务，公司赢得了广大客户的信赖与支持。在雅安地区，公司已树立了良好的行业声誉，成为众多单位的首选合作伙伴。
目前，公司与以下单位建立了长期战略合作关系：
• 中国长安汽车集团股份有限公司四川建安车桥分公司
• 中国移动通信集团四川有限公司雅安分公司
• 中国人民解放军78511部队（驻雅安）
• 四川省雅安电力（集团）股份有限公司
• 雅安市邮政银行
• 雅安市地税局
• 雅安市公路局
作为联想电脑公司在雅安地区的指定经销商，公司更是获得多项荣誉资质：
雅安全星级联想专卖店
雅安地区家电下乡指定店面
